Detailed Description

API-Controller für Sitzungen (Sessions) eines bestimmten Programms (LogEntry).

  • Erlaubt das Anlegen neuer Sessions für ein LogEntry.
  • Nutzt Entity Framework Core für Datenbankzugriffe.
