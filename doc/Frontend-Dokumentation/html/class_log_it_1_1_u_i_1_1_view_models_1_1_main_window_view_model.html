Detailed Description

ViewModel für das Hauptfenster.

  • Stellt Daten für die UI bereit (Tabellen, Diagramme, Labels)
  • Holt Daten vom Backend
  • Bereitet Daten für LiveCharts2 auf
  • Aktualisiert regelmäßig die Anzeige
